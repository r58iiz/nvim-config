# Neovim Tricks

## Insert mode

| Keybind      | Action                                                   |
|--------------|----------------------------------------------------------|
| `<C-w>`      | Delete word                                              |
| `<C-h>`      | Delete letter                                            |
| `<C-u>`      | Delete till start of line                                |
| `<C-x><C-f>` | File completion                                          |
| `<C-o>`      | Oneshot mode (one normal mode command -> back to insert) |
| `<C-m>`      | Insert newline                                           |
| `<C-g>`      | hjkl movement (single)                                   |
| `<C-r>`      | Insert from registers                                    |
